Accessible & User-Friendly Ad UI Components

Welcome to Project AdUI! This open-source initiative provides a library of pre-designed, accessible, and user-experience-focused UI components for various online advertisement formats, built entirely with standard HTML and CSS.

Our mission is to help developers integrate advertising in a way that respects users, enhances accessibility, and improves overall site performance.

Interstitial/Modal Ad

A fullscreen ad that appears over content. Includes a close button.

Advertisement

This is an example of an interstitial ad overlay. Click the X to close.

HTML Structure:

<div class="adui-interstitial-overlay">
    <div class="adui-interstitial-dialog">
        <button class="adui-interstitial-close" aria-label="Close ad">&times;</button>
        <div class="adui-ad-container">
             <div class="adui-label">Advertisement</div>
             <div class="adui-content">
                 <!-- Your ad creative here -->
                 <img src="your-ad-image.jpg" alt="Description of ad">
             </div>
        </div>
    </div>
</div>

Basic JavaScript for showing/hiding:

<script>
    function showInterstitial() {
        document.getElementById('YOUR_INTERSTITIAL_ID').classList.add('is-visible');
        // Add focus trapping and management for accessibility
    }

    function hideInterstitial() {
        document.getElementById('YOUR_INTERSTITIAL_ID').classList.remove('is-visible');
        // Return focus to triggering element
    }

    // Add event listeners to trigger showInterstitial()
    // Add event listener to the close button to trigger hideInterstitial()
    // Add Escape key listener to trigger hideInterstitial()
</script>

Basic Native Ad Structure

A flexible structure for native ads that blend more naturally with content.

Example Native Ad Title

by Example Advertiser

HTML Structure:

<div class="adui-native-ad">
    <div class="adui-label">Sponsored</div>
    <div class="adui-native-media">
        <!-- Ad image or video -->
        <img src="your-ad-image.jpg" alt="Description of ad media">
    </div>
    <div class="adui-native-title">Your Ad Title</div>
    <div class="adui-native-body">Your ad description text.</div>
    <a href="#" class="adui-native-cta">Call to Action</a>
    <div class="adui-native-attribution">by Advertiser Name</div>
</div>
