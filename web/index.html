Building a Better Web, One Ad at a Time

Welcome to Project AdUI! This open-source initiative provides a library of pre-designed, accessible, and user-experience-focused UI components for various online advertisement formats, built entirely with standard HTML, CSS, and minimal JS.

Our mission is to help developers integrate advertising in a way that respects users, enhances accessibility, and improves overall site performance. Join us in creating a more user-friendly online advertising ecosystem.

Interstitial/Modal Ad

A fullscreen ad that appears over content. Includes a clear close button and basic JavaScript for control and accessibility.

Advertisement

This is an example of an interstitial ad overlay. Click the X or press Escape to close.

HTML Structure (`components/interstitial.html`):

<div class="adui-interstitial-overlay" id="YOUR_UNIQUE_INTERSTITIAL_ID">
    <div class="adui-interstitial-dialog" role="dialog" aria-modal="true" aria-labelledby="adui-interstitial-title" tabindex="-1">
        <button class="adui-interstitial-close" aria-label="Close ad">&times;</button>
        <div class="adui-ad-container">
             <div class="adui-label">Advertisement</div>
             <div class="adui-content">
                 <!-- Your ad creative here -->
                 <a href="YOUR_AD_DESTINATION_URL" target="_blank" rel="noopener noreferrer">
                     <img src="your-ad-image.jpg" alt="Description of ad">
                 </a>
                 <h2 id="adui-interstitial-title" class="sr-only">Advertisement</h2>
             </div>
        </div>
    </div>
</div>

Required JavaScript (`script.js`):

<script src="path/to/script.js"></script>
<script>
    // Example usage:
    // window.showInterstitial();
    // window.hideInterstitial();
</script>

Basic Native Ad Structure

A flexible structure for native ads that can blend more naturally with surrounding content, including media, title, body, CTA, and attribution.

Sponsored
Example Native Ad Title
This is a brief description of the sponsored content. It should be relevant and engaging to the user.
Learn More
by Example Advertiser

HTML Structure (`components/native.html`):

<div class="adui-native-ad">
    <div class="adui-label">Sponsored</div>
    <div class="adui-native-media">
        <!-- Ad image or video -->
        <a href="YOUR_AD_DESTINATION_URL" target="_blank" rel="noopener noreferrer">
            <img src="your-ad-image.jpg" alt="Description of ad media">
        </a>
    </div>
    <div class="adui-native-title">Your Ad Title</div>
    <div class="adui-native-body">Your ad description text.</div>
    <a href="YOUR_AD_DESTINATION_URL" target="_blank" rel="noopener noreferrer" class="adui-native-cta">Call to Action</a>
    <div class="adui-native-attribution">by Advertiser Name</div>
</div>

Video Ad Container

A structure for embedding video ads, including a responsive container for the video player and an optional overlay for labels or CTAs.

Advertisement
Advertisement
Watch Now

Placeholder for video ad.

HTML Structure (`components/video-container.html`):

<div class="adui-ad-container">
    <div class="adui-label">Advertisement</div>
    <div class="adui-content">
        <div class="adui-video-container">
            <!-- Your video player code goes here (e.g., <video>, <iframe>) -->
            <video controls>
                 <source src="your-video.mp4" type="video/mp4">
                 Your browser does not support the video tag.
            </video>
            <!-- Optional: Overlay for ad elements like label or CTA -->
            <div class="adui-video-overlay">
                 <div class="adui-label">Advertisement</div>
                 <a href="YOUR_AD_DESTINATION_URL" target="_blank" rel="noopener noreferrer" class="adui-video-cta">Call to Action</a>
            </div>
        </div>
         <!-- Optional: Additional ad text below video -->
         <p class="mt-2 text-center text-sm text-gray-600">Learn more about this offer.</p>
    </div>
</div>
